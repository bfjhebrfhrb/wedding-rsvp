# 🤖 Настройка Telegram уведомлений

## Что это даёт?

При каждой новой RSVP заявке вы будете получать красиво оформленное уведомление в Telegram с полной информацией о госте:
- ✅ Имя гостя
- 📝 Подтверждение присутствия
- 🍷 Предпочтения по напиткам
- 📅 Участие во втором дне
- 🏨 Необходимость размещения
- 👶 Информация о детях
- 🎵 Музыкальные пожелания
- ⚠️ Аллергии и ограничения

---

## 📋 Пошаговая инструкция

### Шаг 1: Создайте Telegram бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду: `/newbot`
3. Введите имя бота (например: `Wedding RSVP Notifier`)
4. Введите username бота (например: `your_wedding_rsvp_bot`)
5. **Скопируйте токен** — он выглядит так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

### Шаг 2: Получите ваш Chat ID

**Вариант А: Через бота**
1. Найдите [@userinfobot](https://t.me/userinfobot) в Telegram
2. Нажмите **Start**
3. Бот отправит ваш Chat ID (например: `123456789`)

**Вариант Б: Через API**
1. Напишите что-нибудь вашему новому боту
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
3. Найдите `"chat":{"id":123456789}` в ответе

### Шаг 3: Настройте переменные окружения на Render

1. Откройте [Render Dashboard](https://dashboard.render.com/)
2. Выберите ваш Web Service `wedding-rsvp`
3. Перейдите в **Environment** (в левом меню)
4. Нажмите **Add Environment Variable**
5. Добавьте две переменные:

   **Первая переменная:**
   - **Key**: `TELEGRAM_BOT_TOKEN`
   - **Value**: `8740040801:AAFkpmQ6OqijO-AacYPPzizComEkjLKYeSs`
   
   **Вторая переменная:**
   - **Key**: `TELEGRAM_CHAT_ID`
   - **Value**: `869911026`

6. Нажмите **Save Changes**

### Шаг 4: Перезапустите сервис

Render автоматически перезапустит сервис после сохранения переменных окружения.

Если этого не произошло:
1. Перейдите на вкладку **Manual Deploy**
2. Нажмите **Deploy latest commit**

1. Откройте ваше приглашение: `https://your-app.onrender.com/`

## 🔧 Локальное тестирование

Для тестирования на локальной машине:

1. Создайте файл `.env` в корне проекта:
   ```bash
   TELEGRAM_BOT_TOKEN=8740040801:AAFkpmQ6OqijO-AacYPPzizComEkjLKYeSs
   TELEGRAM_CHAT_ID=869911026
   ```

2. Установите python-dotenv (опционально):
   ```bash
   pip install python-dotenv
   ```

3. Запустите сервер:
   ```bash
   python server.py
   ```

## 🛠️ Устранение неполадок

### Уведомления не приходят

1. **Проверьте переменные окружения:**
   - Убедитесь, что `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` правильно заданы в Render
   - Проверьте, что нет лишних пробелов в начале/конце значений

2. **Проверьте логи Render:**
   - Откройте вкладку **Logs** в Render Dashboard
   - Найдите строку: `✅ Telegram уведомления активированы для chat_id: 869911026`
   - Если видите `⚠️ Telegram уведомления отключены` — переменные не заданы

3. **Проверьте бота:**
   - Убедитесь, что вы написали боту хотя бы одно сообщение (нажали Start)
   - Бот не может отправлять сообщения, пока вы не инициируете диалог

4. **Проверьте токен:**
   - Токен должен быть активным
   - Если создали нового бота, используйте новый токен

### Бот отправляет сообщения, но они не форматированы

Это нормально — HTML форматирование может не работать в некоторых случаях. Главное, что информация приходит.

## 📱 Дополнительные возможности

### Отправка в группу

Если хотите, чтобы уведомления приходили в группу:

1. Создайте группу в Telegram
2. Добавьте вашего бота в группу
3. Сделайте бота администратором
4. Получите Chat ID группы (он будет отрицательным, например: `-123456789`)
5. Используйте этот ID в `TELEGRAM_CHAT_ID`

### Несколько получателей

К сожалению, текущая реализация поддерживает только один Chat ID. Если нужно отправлять нескольким людям, используйте группу.
